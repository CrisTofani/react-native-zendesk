# @pagopa/react-native-zendesk

A react-native wrapper for Zendesk Unified SDK

## Installation

```sh
npm install @pagopa/react-native-zendesk
```

## Usage

```js
import { multiply } from '@pagopa/react-native-zendesk';

// ...

const result = await multiply(3, 7);
```
